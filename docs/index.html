Synthetic data that respects your schema

Concoction discovers your database, generates realistic relational data with full referential integrity, and exports it in CSV, JSON, and SQL — deterministically, every time.

Get Started Read the Docs GitHub 
# Discover schema, generate 100 rows, export CSV + JSON + SQL
dotnet run --project ./Concoction.Cli -- generate \
  --provider    sqlite \
  --connection  "Data Source=./mydb.db" \
  --seed        42 \
  --rows        100 \
  --rules       ./rules.yaml \
  --output      ./artifacts

🔍

Schema-Aware

Automatically discovers tables, columns, primary keys, foreign keys, unique constraints, and indexes from live SQLite and PostgreSQL databases.

🎲

Deterministic

Every generation run is reproducible. Pass --seed 42 and get the same dataset every time — ideal for CI/CD pipelines and fixture snapshots.

🔗

Referential Integrity

Topological sort ensures parent tables are populated before child tables. FK values always reference real rows. Self-referencing tables get a safe chain structure.

📋

Rules DSL

Override generation strategies with a YAML or JSON rules file. Pin fixed values, set null rates, apply weighted distributions, and target nested JSON paths.

🌐

REST API

Full multi-tenant REST API built with ASP.NET Core Minimal API. Accounts, workspaces, projects, runs, workflows, chat, and API key management — all behind Swagger UI.

📦

TypeScript SDK

Install @concoction/client for a fully-typed client that covers every REST endpoint, with pollRun for long-running workflow tracking.
