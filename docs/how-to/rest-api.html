REST API Reference

ASP.NET Core Minimal API. Listens on port 5000 by default. Swagger UI at /swagger.

Authentication

All endpoints require the X-Api-Key header:

X-Api-Key: cnc_yoursecrethere

Missing or invalid keys return 401 Unauthorized.

Rate Limiting

Fixed window: 100 requests per minute. Exceeding the limit returns 429 Too Many Requests.

Swagger UI

http://localhost:5000/swagger — all routes described with OpenAPI annotations.

API Keys

POST /accounts/{accountId}/api-keys

POST /accounts/3fa85f64-.../api-keys
{
  "name": "ci-pipeline",
  "scopes": ["workspace:read", "workspace:write"],
  "expiry": "90.00:00:00"
}

200 OK
{
  "id": "3fa85f64-...",
  "name": "ci-pipeline",
  "plaintextSecret": "cnc_abc123...",
  "scopes": ["workspace:read","workspace:write"],
  "expiresAt": "2024-09-01T00:00:00Z"
}

The plaintextSecret is shown once only.

GET /accounts/{accountId}/api-keys

Returns metadata only (no secrets).

DELETE /accounts/{accountId}/api-keys/{keyId}

Revokes the key immediately. Returns revoked key with isRevoked: true.

Error Responses

All errors use RFC 7807 Problem Details format:

{
  "type": "https://tools.ietf.org/html/rfc7231#section-6.5.4",
  "title": "Not Found",
  "status": 404,
  "detail": "Account 3fa85f64-... was not found."
}
StatusMeaning
400Bad request (validation failed)
401Missing or invalid API key
403Insufficient scopes
404Resource not found
409Conflict (e.g. cancel a completed run)
429Rate limit exceeded
500Internal server error

Starting the API Locally

dotnet run --project ./Concoction.Api/Concoction.Api.csproj

Default: SQLite at concoction.db in the working directory. Configure PostgreSQL via appsettings.json or environment variables.
