TypeScript SDK

The @concoction/client npm package provides a fully-typed client for the Concoction REST API. Ships as CJS + ESM + TypeScript declaration files.

Installation

npm install @concoction/client

Or build from source:

cd sdk/typescript
npm install
npm run build

Quick Start

import { ConcoctionClient, ConcoctionError } from "@concoction/client";

const client = new ConcoctionClient({
  baseUrl: "http://localhost:5000",
  apiKey:  "cnc_yoursecrethere",
});

const account   = await client.createAccount("Acme Corp");
const workspace = await client.createWorkspace(account.id, "Production Clone");
const project   = await client.createProject(workspace.id, "Orders Dataset");

Constructor Options

interface ConcoctionClientOptions {
  /** Base URL of the Concoction API */
  baseUrl: string;
  /** API key sent as X-Api-Key header */
  apiKey: string;
  /** Optional custom fetch — defaults to globalThis.fetch */
  fetch?: typeof globalThis.fetch;
}

Accounts

createAccount(name)

const account = await client.createAccount("Acme Corp");
// { id: "3fa85f64-...", name: "Acme Corp", createdAt: "2024-06-01T10:00:00Z" }

getAccount(accountId)

const account = await client.getAccount("3fa85f64-...");

listMembers(accountId)

const members = await client.listMembers(account.id);
// [{ accountId, userId, role: "Owner"|"Member", joinedAt }]

inviteUser(accountId, email, expiryHours?)

await client.inviteUser(account.id, "bob@example.com", 48);

Dataset Runs

listRuns(projectId, page?, pageSize?)

const page = await client.listRuns(project.id, 1, 20);
// { items: [...], totalCount: N, page: 1, pageSize: 20 }

getRun(runId)

const run = await client.getRun("run-abc123");
// status: "Queued"|"Running"|"Completed"|"Failed"|"Cancelled"

cancelRun(runId)

await client.cancelRun("run-abc123");

pollRun(runId, options?)

Polls until terminal state. Throws ConcoctionError on failure, cancellation, or timeout.

const { runId } = await client.runWorkflow(workflow.id);
const completed = await client.pollRun(runId, {
  intervalMs: 3000,    // default 2000
  timeoutMs:  300_000, // default 120000
});
console.log("Completed at:", completed.completedAt);

Error Handling

try {
  const account = await client.getAccount("non-existent");
} catch (err) {
  if (err instanceof ConcoctionError) {
    console.error(HTTP +""+\+""+: +""+\+"");
    console.error(err.detail);
  }
}
